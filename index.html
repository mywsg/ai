Automa

What is it?

  • A low-code/no-code browser extension to replace repetitive manual web-based tasks.
  • Works on standard browsers like Chrome, Firefox and Edge.

How to Use It

  • Install the free Automa extension inside your browser.
  • Go to the Automa dashboard, click "New Workflow". then drag-and-drop blocks to build your workflow.
  • Save your workflow, then run it.
  • When in doubt, prompt Gen AI for help to walk you through how to use Automa.

Data Security & Governance

  • Automa's workflow only runs on your laptop browser. It does not transmit information in downloaded files to external hosts.
  • Configure Automa to run only on selected URLs.
  • Always log in manually to a website before you run Automa's workflow.

Power Automate

What is it?

  • A low-code/no-code app that connects different apps & services within Microsoft 365.
  • You tell it to spot an action (e.g. change in a file), then carry out a sequence of other actions, then it repeats the steps every time.
  • You can run it manually when needed, or schedule runs.

How to Use It

  • Log in to your corporate M365 account and select Power Automate.
  • Create a cloud flow using an initial trigger, logic blocks and follow-on actions.
  • When in doubt, prompt Gen AI for help to walk you through how to use Power Automate.

Data Security & Governance

  • The app is included within MYW’s M365 free Education license.
  • It only connects to MYW M365 business apps, which keeps data secure.

Visual Basic for Applications (VBA)

What is it?

  • Microsoft's event-driven programming language built into Excel, which is used to programme Macros.

How to Use It

  • Tell Gen AI how you want to format your data and ask it to create a VBA code.
  • Set up your file according to what Gen AI tells you.
  • Paste the code in the Developer tab > VBA, then run the macro.
  • When in doubt, prompt Gen AI for help to walk you through how to use VBA.

Data Security & Governance

  • VBA runs within files in your laptop. It does not transmit data to external hosts.
  • The Developer tab is not available in the M365 free Education license. You have to upgrade your license to use this tool. Request upgrade to M365 Premium through Pillar Heads → Pillar Heads to put up request to MIS.

Google Apps Script (GAS)

What is it?

  • Built-in engine developed by Google to automate tasks and build webpages for Google-based files.

How to Use It

  • Tell Gen AI how you want to format your data and ask it to create a GAS code.
  • Open your Google Sheet and click *Extensions > Apps Script* from the top header menu.
  • Paste the code in the tab, then deploy it.
  • When in doubt, prompt Gen AI for help to walk you through how to use GAS.

Data Security & Governance

  • Restrict GAS permission to read data only on the file you are using, to protect data contained in other Google Drive files.
